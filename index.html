Undergraduate Thesis

    My undergraduate thesis as a student in the University of Utah’s Honors College focused on answering the question: how will governments regulate automotive cybersecurity? As an emerging trend, I sought to define such risks to user safety and privacy, the current regulatory framework, cybersecurity regulation timelines during other industries’ development, case studies, and ultimately recommendations. With such severe implications for owners of connected vehicles and any road user, I found that there is a significant need for widespread and standardized regulation.

Simple Website

I created a simple website using HTML, PHP, and CSS as a proof-of concept for page authorization based on user role, input sanitization, and storing passwords as salted hashes rather than plaintext in a SQL database. Each function connects with the SQL database to update records within entities in real-time.
